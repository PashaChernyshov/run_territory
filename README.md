# Map Nowoe

Flutter web-проект: карта для афиши.

## Локальный запуск

1. Установить зависимости:

```bash
flutter pub get
```

2. Запустить web-server:

```bash
flutter run -d web-server --web-hostname 0.0.0.0 --web-port 8080
```

3. Открыть в браузере на ПК:

```text
http://localhost:8080
```

## Актуальные мероприятия (web)

Для загрузки событий в браузере нужен локальный прокси (обходит CORS и кэширует на 10 минут):

```bash
dart run tool/events_proxy.dart
```

Прокси стартует на `http://0.0.0.0:8787/events`.

Проверка с телефона в той же Wi-Fi сети:

1. Узнать IP ПК в локальной сети (например `192.168.1.42`).
2. Открыть сайт на телефоне: `http://192.168.1.42:8080`.
3. Прокси должен быть запущен на ПК командой выше (приложение само пойдёт в `http://192.168.1.42:8787/events`).
